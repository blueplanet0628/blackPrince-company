title: Python / C++ 業務自動化・ツール開発

Python・C++を用いた業務支援ツール、スクレイピング、API連携、デスクトップアプリの開発経験あり。

 

業務効率化や自動化を目的としたツール・システムをPythonやC++で開発してきました。スタンドアロンツールからAPI連携、自動レポート生成まで、現場のニーズに応じたソリューションを提供しています。

【開発実績（一部抜粋）】
Webスクレイピングツール（Python）
　└ BeautifulSoup、Selenium、Requestsによる定期データ取得とCSV出力
　└ スケジューラ連携での自動実行

自動レポート作成システム（Python）
　└ データ集計→Excel/PDF化→メール送信まで一括自動処理

API連携ツール（Python）
　└ Chatwork / Slack / Google Sheets / Notion などへの自動投稿や双方向同期

業務用スタンドアロンツール（C++）
　└ Windows上で動作する高速なファイル変換ツール
　└ USBデバイス制御用アプリケーションの設計・開発

センサー連携・リアルタイム処理（C++）
　└ シリアル通信を用いたリアルタイムセンサーデータ処理システム

【使用技術・ツール】
言語：Python / C++ / Shell

ライブラリ：Selenium / BeautifulSoup / pandas / OpenPyXL / Qt

OS：Windows / Linux

その他：Git / REST API / Webhook / cron / VSCode / CMake
